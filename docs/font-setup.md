# تنظیمات Font در پروژه

## فونت Sahel

فونت Sahel به صورت محلی در مسیر `public/fonts/` ذخیره شده تا از وابستگی به CDN و هشدارهای شبکه جلوگیری شود.

### فایل‌های فونت:
- `Sahel-Regular.woff2` - وزن 400 (نرمال)
- `Sahel-SemiBold.woff2` - وزن 600 (نیمه‌ضخیم)
- `Sahel-Bold.woff2` - وزن 700 (ضخیم)

### نحوه استفاده:
فونت Sahel در `user.css` برای تمام عناصر تنظیم شده و از `font-display: swap` استفاده می‌کند تا از مشکلات FOUT جلوگیری شود.

### تنظیمات Font-face:
در `layout.blade.php` تعریف شده و شامل:
- پشتیبانی از فرمت WOFF2 برای بهترین فشرده‌سازی
- font-display: swap برای تجربه بهتر کاربر
- وزن‌های مختلف برای استفاده در UI

## نکات مهم:
- فونت‌ها به صورت محلی لود می‌شوند (بدون CDN)
- از font-display: swap استفاده شده
- آیکون‌های Font Awesome مستثنی هستند
